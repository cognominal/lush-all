# Language Studio

## incrementally posh a exising language or a new one

Early spec:

Language Studio proposes a framework  
to help to transform a vanilla susy to a poshed and huffed one.

It provides sample vanilla sources files you can select by type.

When a file is selected, 
It calls the corrext parsers

The file is displayed, its posh susy is updated
in lushed. The AST yamls are displayed for the vanilla and the posh susys.
